# Dashboard Analytics for MBG Sentiment Analysis

Folder ini berisi file-file untuk dashboard analytics yang digunakan untuk visualisasi data sentimen dari program Makan Bergizi Gratis (MBG).

## Isi Folder:
- File-file PowerBI untuk visualisasi data sentimen
- Template dashboard yang dapat disesuaikan
- Spesifikasi untuk integrasi data dari MongoDB

## Cara Menggunakan:
1. Buka file `.pbix` dengan PowerBI Desktop
2. Hubungkan ke data yang dihasilkan dari proses ETL
3. Sesuaikan visualisasi sesuai kebutuhan
